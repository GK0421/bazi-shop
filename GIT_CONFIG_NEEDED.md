# Git 同步说明

当前仓库远端：

`https://github.com/GK0421/bazi-shop.git`

如果本机还没设置 Git 用户信息，可执行：

```powershell
git config --global user.name "你的 GitHub 用户名"
git config --global user.email "你的 GitHub 邮箱"
```

然后在项目目录提交：

```powershell
cd D:\Develop\bazi-shop
git add .
git commit -m "feat: rebuild project into crazy abyss douyin game"
git push origin master
```
