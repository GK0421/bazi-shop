# 疯狂深渊上线清单

## 当前已完成

- 抖音小游戏入口、`AppID` 和 Canvas 主循环已打通
- 开始界面支持选择 4 名调查员
- 9 房间推进、Boss 房、祭坛房、安全屋流程已跑通
- 疯狂值、技能、换人、武器、消耗品、Buff 骨架已接入
- 暂停层、教程层、结算页疯狂曲线已补齐
- 分享入口与侧边栏复访入口已加入代码结构
- 本地辅助服务和音频生成脚本已经放入仓库
- 文档、工程名和目录结构已统一为《疯狂深渊》

## 上线前仍需人工补齐

- 在抖音开放平台填入真实能力配置：
  - 插屏广告位 ID
  - 激励广告位 ID
  - 订阅消息模板 ID
  - 内购商品与资质配置
- 在开发者工具内完成：
  - 真机预览
  - 分包 / 包体检查
  - 重新执行审核预检
  - 上传代码和体验版验证
- 补正式资源：
  - 像素角色帧动画
  - 敌人与 Boss 动画
  - UI 图标与按钮态
  - 宣传封面、图标、录屏视频
- 补正式音频：
  - 替换当前程序化占位音效
  - 生成或采购可发布 BGM / SFX

## 我建议的上线顺序

1. 先用当前版本在抖音开发者工具做完整真机试玩。
2. 把广告、订阅、内购的真实平台 ID 全部填进配置。
3. 确认预检通过后，再补最终美术和可发布音频。
4. 最后做一轮连续 20 局以上的长测，重点看卡顿、内存、误触和音频。

## 当前版本定位

- 代码完成度：可提审 beta
- 内容完成度：玩法闭环已成型
- 商业化完成度：需要平台真实配置
- 资源完成度：仍需正式美术与音频
